# Contributing to Python Calculator

Thank you for considering contributing! Here's how to get started.

## How to Contribute

1. Fork the repository
2. Create a new branch: `git checkout -b feature/your-feature-name`
3. Make your changes
4. Commit: `git commit -m "Add: brief description of your change"`
5. Push: `git push origin feature/your-feature-name`
6. Open a Pull Request

## What You Can Contribute

- Bug fixes
- New features (e.g. history log, keyboard shortcuts, scientific mode)
- UI improvements
- Code cleanup or refactoring
- Documentation updates

## Code Guidelines

- Follow PEP 8 style for Python code
- Keep the PyQt5 structure consistent with the existing codebase
- Test your changes before submitting
- Write clear, descriptive commit messages

## Reporting Bugs

Open an issue and include:
- What you expected to happen
- What actually happened
- Steps to reproduce it
- Your Python and PyQt5 version

## Questions?

Open an issue and tag it with the `question` label.

# 🧮 Python Calculator

A desktop calculator app built with Python and PyQt5, styled with a sleek dark theme inspired by the iOS Calculator.

## Features
- Basic operations: addition, subtraction, multiplication, division
- Percentage calculation (context-aware with operators)
- Toggle positive/negative sign (+/-)
- Backspace and clear (AC) support
- Division by zero error handling
- Dynamic font scaling based on number length
- Responsive hover effects
